JavaScript: Javascript is a single-threaded programming language.It executes a script from the top of the file and works its way down. It is a client-side scripting language or browser scripting.Client-side scripting language means that the browser will run / execute these scripts.Javascript Basically work synchronize way.Javascript is JIT-Just in Time Language.

Local Storage: With local storage, web applications can store data locally within the user's browser.Local storage stores data with no expiration date. Session Storage: Session storage is destroyed once the user closes the browser whereas. Changes are available per window.

The event loop is the secret by which JavaScript gives us an illusion of being multithreaded even though it is single-threaded.An event loop is something that pulls stuff out of the queue and places it onto the function execution stack whenever the function stack becomes empty
